## [1.0.0] 15-12-2025

Not the first update nor commit, but the official 1.0 version of my portfolio!

### Added

  • Visual Studio Code inspired UI with working side- and tab bar.
  • Mobile support! This portfolio is best viewed on larger screens like a laptop, but it also works on mobile.
    Mobile related features:
    • Togglable sidebar (also works on larger screens).
    • Automaticly toggle sidebar off when opening or selecting a file.
  • Website wide save state. This uses local storage so it can be deleted by the usee at any time.
    It saves:
    • Which folders are open and which are closed.
    • The currently selected file.
    • The selected theme. Color theme can be changed in setting.
  • Animated "3d" folder icons :P.
