Uomo Webshop

This project is a responsive, multi-page webshop created for a fictional pharmaceutical brand called Uomo. It was developed as the final assignment for a CSS course, with the main goal of learning how to accurately translate a provided design into a fully responsive website.

The entire layout and visual style are based on an existing Figma design , focusing on consistent spacing, typography, and responsive behavior across different screen sizes.

This assignment was completed as part of the CSS course on CodeCrashers, and demonstrates my ability to work from real design files, structure CSS efficiently, and build layouts that adapt smoothly to different devices.
