Uomo Webshop

Uomo is a fully responsive, multi-page front-end project for a fictional pharmaceutical webshop. The goal of the project was to accurately translate a provided design into a maintainable and scalable website using modern CSS techniques.

The layout and visual system are based on a detailed Figma design , with a strong focus on consistent spacing, typography, component alignment, and breakpoint behavior.

The project is structured around reusable layout patterns, CSS custom properties, and clearly separated components, making the design easy to extend or adapt. Interactive elements such as product cards, promotional blocks, and navigation controls were carefully implemented to match the original design intent.

This project was completed as part of the CSS course at CodeCrashers and showcases my ability to work from professional design files and convert them into clean, responsive front-end code.
